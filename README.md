# Landing Page

Este proyecto es una página de aterrizaje simple que incluye HTML, CSS y JavaScript. A continuación se detallan los archivos incluidos en el proyecto y sus funciones.

## Estructura del Proyecto

- `index.html`: Este archivo es la estructura principal de la página de aterrizaje. Contiene la estructura básica de HTML, incluyendo el encabezado, el cuerpo y enlaces a los archivos CSS y JS.
  
- `styles.css`: Este archivo contiene los estilos para la página de aterrizaje. Define el diseño, colores, fuentes y otros estilos visuales para los elementos HTML.
  
- `script.js`: Este archivo contiene el código JavaScript para la página de aterrizaje. Se encarga de la interactividad, como manejar eventos y manipular el DOM.

## Instrucciones para Ejecutar

1. Clona este repositorio en tu máquina local.
2. Abre el archivo `index.html` en tu navegador web.
3. Asegúrate de que los archivos `styles.css` y `script.js` estén en la misma carpeta que `index.html` para que se carguen correctamente.

## Tecnologías Utilizadas
